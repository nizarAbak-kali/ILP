Ce regroupement de paquetages correspond au langage ILP1. Il est divisé logiquement en:

  • Un paquetage fromxml pour lire les programmes et les traduire en un arbre de syntaxe abstraite (AST).
  • Les AST sont décrits par un arbre d'interfaces interfaces.
  • Un paquetage eval pour évaluer (interpréter) l'AST. Ce paquetage a besoin d'une bibliothèque d'exécution qui se trouve en runtime.
  • Un paquetage cgen pour compiler l'AST en un programme C. La bibliothèque d'exécution est bien sÃ»r en C (et dans le répertoire C :).
